This wizard will guide you through the configuration of your Scale-Out Computing on AWS (SOCA) cluster. This tool does not actually install SOCA but simply generates a CloudFormation URL with all your inputs. You can then import it in your AWS CloudFormation console to proceed to the actual installation of SOCA.

I want to use my existing resources


Pre-Requisites
You must have built and uploaded your Scale-Out Computing on AWS cluster on S3 bucket.
You must have a VPC
Your VPC must have 6 subnets (3 private/3 public)
You must have 2 Elastic File Systems
You must have one SSH keypair
You must have two security groups configured correctly
What resources SOCA will still create:
  • The EC2 instance used as Scheduler machine
  • 3 IAM roles
  • 1 Lambda function
  • 1 Elastic IP (EIP)
