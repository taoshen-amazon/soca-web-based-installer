This website will help you configure SOCA Cluster with your existing resources

Pre-Requisites

You must have built and uploaded your Scale-Out Computing on AWS cluster on S3 bucket
You must have a VPC
Your VPC must have 6 subnets (3 private/3 public)
You must have 2 Elastic File Systems
You must have one SSH keypair

What resources SOCA will still create:

  • The EC2 instance used as Scheduler machine
  • 2 security groups and 3 IAM roles
  • 1 lambda function
  • 1 Elastic IP (EIP)
